## 五邑大学课程表app

原生Android实现

## 功能

- 数据爬取服务端在 https://github.com/novohit/wyu-lesson (SpringBoot)

- 启动页

- 登录

- 每周课表（高亮显示当天）
- 今日课表

- 成绩查询
- 考试安排
- ChatGPT聊天
